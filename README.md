# Mac-setup

## 1 Install Homebrew

```sh
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

## 2 install Brewfile

```sh
git clone https://github.com/MattBidewell/mac-setup
cd mac-setup
brew bundle
```

## 3 Add dotfiles

```bash
bash <(curl -fsSL ...)
```
